  6. Develop algorithms using dynamic variables and pointers to construct a singly linked list consisting of Student ID (integer), Student Name (character string), Semester (integer). The operations to be supported are:
    1. a. Inserting a node
      1. At the front of a list
      2. At the rear of the list
      3. At any position in the list
    2. b. Deleting a node based on Student ID (display an error message if not found)
    3. c. Searching a node based on Student ID (display an error message if not found)
    4. d. Displaying all the nodes in the list
    Code in C++.
  7. Develop an algorithm using dynamic variables and pointers to construct a stack of integers using singly linked list and perform:
    1. a. Push
    2. b. Pop
    3. c. Display
    The program should print appropriate messages for stack overflow and stack empty. Code in C++.
  8. Develop an algorithm using dynamic variables and pointers to construct a queue of integers using singly linked list and perform:
    1. a. Insert
    2. b. Delete
    3. c. Display
    The program should print appropriate messages for queue full and queue empty. Code in C++.
  9. Develop an algorithm to support operations on a doubly linked list where each node consists of an integer data object:
    1. a. Create a doubly linked list
    2. b. Insert a new node
    3. c. Delete a specific node
    4. d. Display the contents of the list
    Code in C++.
  10. Develop algorithms to:

    2. Traverse the binary tree using:
      1. a. Inorder (recursive & non-recursive)

      3. c. Postorder (recursive & non-recursive)
    3. Display the elements in the tree
    Code in C++.
  11. Develop algorithms to create a Binary Search Tree (BST) and perform:

    4. d. Find predecessor of a node
    5. e. Find successor of a node
    6. f. Delete a node with a specific key value
    Code in C++.
  12. Develop an algorithm to construct an AVL tree for a given set of elements. Code in C++.
  13. Develop algorithms to sort a given list of elements (numbers or strings):
    1. a. Insertion Sort (C)
    2. b. Merge Sort (C)
    3. c. Quick Sort
    4. d. Heap Sort
